# MercuryStart
人人都是墨丘利，可以快速开始

本项目基于vue3采用技术栈
- Vue3
- Vite
- Element-plus
- pinia
- mitt
- vue-type
- axios

# 开发规划
1. 完成基础组件的配置
  1. 路由跳转
  2. 权限控制
  3. 组件间三种数据传输模式
2. 完成常见基础组件开发，即对elplus进行重组
  1. el-table+el-page
  2. mutifilter
  ...
